## 🛠️ Customization

### 1. Update Videos
Open `index.html` and search for the `<iframe>` tags. I have already set them to your provided link `yCW-1EFvp2A`.

### 2. Paste Your Full Indicator Code
Locate the `#locked-1` (Scalping) and `#locked-2` (Strategy) sections in `index.html`. Replace the placeholder Pine Script code inside the `<pre>` tags with your full source code.

### 3. Setup Payment Button
The "Unlock" buttons currently scroll to the code sections. Replace the `href="#pricing"` in the locked sections or the links in the pricing cards with your actual payment gateway URLs (e.g., Razorpay, PayPal).

## 🌐 Deploying to GitHub Pages
1. Create a new repository on GitHub.
2. Upload the `index.html`, `README.md`, and the `hero_trading_chart_background_*.png` file.
3. Go to **Settings > Pages > Build and deployment**.
4. Select the `main` branch and click **Save**.
5. Your premium landing page will be live instantly!
